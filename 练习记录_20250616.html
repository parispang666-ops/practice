练习记录 · 页面 PRD

页面类型：管理后台类（PC 端） 所属模块：考试中心
变更类型：优化（已有页面，兼容新模式） 最后更新：2026-07-01

1. 引言

1.1 页面概述

练习记录是管理后台「考试中心」模块下的已有页面，原先仅支持专业题目练习（自由练习）模式的记录统计。本次新增模拟练习模式后，页面需兼容两种练习模式的数据，将同一员工的自由练习记录与模拟练习记录合并统计展示在同一个列表中。

管理员可通过姓名、专业、部门进行筛选，支持导出和分页浏览。

1.2 变更说明

对比项原页面（变更前）新页面（变更后）
数据来源仅「专业题目练习」模式新增「专业题目练习」+「模拟练习」两种模式
练习次数专业练习次数专业练习次数 + 模拟练习次数（合并）
练习题数专业练习题数专业练习题数 + 模拟练习题数（合并）
首次练习时间专业练习的首次时间两种模式中最早的时间
上次练习时间专业练习的上次时间两种模式中最晚的时间
其他—其余筛选、导出、分页功能保持不变

1.3 使用角色

角色访问权限
超级管理员✓ 全量数据，可筛选 + 导出
安全管理员✓ 按组织隔离，可筛选 + 导出
普通员工✗（不可访问管理后台页面）

1.4 前置条件

  • 用户已登录管理后台
  • 用户具有「考试中心 → 练习记录」菜单权限
  • 系统中存在练习数据（专业练习或模拟练习已被员工参与）

4. 功能说明

4.1 筛选区

条件控件默认值说明
姓名文本输入框请输入支持模糊搜索
专业下拉选择器请选择专业单选
部门下拉选择器请选择单选

操作按钮：查询（按条件刷新表格）重置（清空条件恢复默认）

4.2 导出

表格卡片顶部左侧「导出」按钮，将当前筛选条件下的全部数据导出为 Excel 文件（不受分页限制）。

4.3 数据表格

数据合并规则（按员工维度聚合）：

合并字段合并规则
练习次数专业练习次数 + 模拟练习次数（累加）
练习题数专业练习题数 + 模拟练习题数（累加）
首次练习时间取两种模式中最早的时间
上次练习时间取两种模式中最晚的时间

表格列：姓名、部门、所属专业、练习次数、练习题数、首次练习时间、上次练习时间、列设置（齿轮图标）

行交互：鼠标悬停高亮；排序：按原有字段排序；列设置：表头最右列齿轮图标，弹出显示/隐藏面板

4.4 分页

「共 N 条」| 每页条数下拉（15/20/50/100）| < > 翻页 | 数字页码按钮 | 跳转输入框 +「前往」+「页」

5. 交互设计

  • 页面加载：自动加载第 1 页，15 条/页，无筛选条件；加载中展示加载态
  • 查询：点击查询重新请求第 1 页；点击重置清空条件并自动查询
  • 导出：按当前筛选条件导出全量数据为 Excel，导出成功后浏览器自动下载
  • 分页：切换每页条数时重置为第 1 页；跳转仅允许正整数，超范围自动纠正

6. 数据模型

6.1 练习记录数据结构（聚合视图）

字段类型说明
员工ID文本唯一标识
姓名文本员工姓名
部门文本所属部门名称
所属专业文本所属专业名称
练习次数数字专业练习次数 + 模拟练习次数
练习题数数字专业练习题数 + 模拟练习题数
首次练习时间日期时间取两者中较早者
上次练习时间日期时间取两者中较晚者

6.2 底层原始记录（按模式分表存储）

专业题目练习记录：记录ID、员工ID、练习模式（专业练习）、练习时间、练习题数

模拟练习记录：记录ID、员工ID、练习模式（模拟练习）、关联练习ID、练习时间、练习题数

6.3 数据保留规则 规则

  • 练习记录与练习配置解耦存储，即使原练习配置被删除，记录仍然保留
  • 即使练习人员被移出练习范围或从系统中删除，其已产生的练习记录仍然保留，不受人员变更影响
  • 仅记录已完成提交的练习结果，未提交的练习不计入统计
  • 专业练习和模拟练习的记录各自独立持久化，通过员工ID关联聚合

6.4 筛选条件数据结构

姓名（文本）、专业（文本）、部门（文本）、当前页（数字，默认 1）、每页条数（数字，默认 15）

7. 异常处理

异常场景处理方式
无数据展示空状态：「暂无数据」
接口请求失败轻提示「加载失败，请稍后重试」
导出失败轻提示「导出失败，请稍后重试」
筛选无结果表格空数据行，分页「共 0 条」
跳转页码超范围自动纠正为最近有效页码
