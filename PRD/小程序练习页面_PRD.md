## 1. 引言

### 1.1 页面概述

> **详见** [整体PRD §7.5 小程序练习页](./练习功能整体PRD.md#75-小程序练习页核心页面)

小程序练习页面是员工进入模拟练习后的核心答题页面。**本页面复用自由练习的练习页面，样式与自由练习完全一致**，题目来源改为试卷（由后台练习配置关联的试卷抽取），所有按钮功能和交互逻辑保持不变。

### 1.2 使用角色

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 从练习列表「进行中」卡片进入 |
| 部门负责人 | ✓ 从练习列表「进行中」卡片进入 |

### 1.3 前置条件

- 从练习列表页点击「进行中」的试卷卡片进入
- 携带练习配置 ID 和名称参数
- 后台根据练习配置从关联试卷中抽取题目

### 1.4 试卷删除与人员变更兼容

> **详见** [整体PRD §7.5 试卷删除与人员变更兼容](./练习功能整体PRD.md#75-小程序练习页核心页面) 及 [整体PRD §10 与现有系统的关系](./练习功能整体PRD.md#10-与现有系统的关系)

- **试卷删除不影响练习**：练习创建时从关联试卷中抽取的题目数据已落库存储，试卷的删除不影响已创建的练习功能与已进入的练习答题。
- **人员变更不影响进行中的练习**：已进入练习中的用户，即使被从练习人员列表中移除，仍可继续当前练习至结束，不会中断答题流程。

## 2. 页面结构

### 2.1 页面布局

```
┌──────────────────────────────┐
│       ··· (状态栏)            │
├──────────────────────────────┤
│  ←  模拟练习                   │  导航栏
├──────────────────────────────┤
│  答题进度              1/4    │
│  ████████░░░░░░░░░░  25%     │  进度头部
├──────────────────────────────┤
│                              │
│  ┌──────────────────────────┐│
│  │ 单选题                    ││  题型标签
│  │                          ││
│  │ 1. 题目内容 ...           ││  题目文字
│  │                          ││
│  │  ○ A. 选项A              ││
│  │  ● B. 选项B              ││  选项列表（选中高亮）
│  │  ○ C. 选项C              ││
│  │  ○ D. 选项D              ││
│  └──────────────────────────┘│
│                              │
├──────────────────────────────┤
│ [答题卡] [提前结束] [上一题] [下一题] │  底部操作栏（固定）
└──────────────────────────────┘
```

### 2.2 导航栏

| 元素 | 位置 | 行为 |
|------|------|------|
| 返回按钮 ← | 左侧 | 弹出确认对话框 → 点击「退出」返回练习列表 |
| 标题 | 居中 | 固定显示「模拟练习」 |

**返回确认弹窗：**
- 标题：确认退出练习
- 内容：退出后将不会保存本次练习，请谨慎操作
- 按钮：取消（关闭弹窗）/ 退出（返回练习列表）

### 2.3 进度头部

| 元素 | 说明 |
|------|------|
| 背景色 | 主题色 |
| 进度文字 | 左侧「答题进度」，右侧「当前题/总题数」 |
| 进度条 | 白色填充条，按 (当前题/总题数) 比例 |

## 3. 题目卡片

### 3.1 题型支持

| 题型 | 标签文案 | 来源 |
|------|:------:|------|
| 单选题 | 单选题 | 试卷题目 |
| 多选题 | 多选题 | 试卷题目 |
| 判断题 | 判断题 | 试卷题目 |

### 3.2 选项交互

| 题型 | 选择方式 |
|------|---------|
| 单选题 | 单选，点击替换上一个选择 |
| 多选题 | 支持多选，点击切换选中/取消 |
| 判断题 | 单选，A 正确 / B 错误 |

选中态高亮显示。

### 3.3 题目来源

题目由后台根据练习配置，从**关联试卷**中按配置的题目数量和题型数量抽取。每次进入练习重新抽题，不保留上次答题进度。

## 4. 底部操作栏

### 4.1 按钮布局

| 按钮 | 位置 | 行为 |
|------|:--:|------|
| 答题卡 | 最左 | 打开答题卡半屏弹窗 |
| 提前结束 | 左二 | 弹出提前结束确认弹窗 |
| 上一题 | 右二 | 回到上一题，可修改答案；首题置灰不可用 |
| 下一题/提交 | 最右 | 进入下一题；最后一题变为「提交」 |

### 4.2 答题卡弹窗（半屏）

> **详见** [整体PRD §7.5 答题卡弹窗](./练习功能整体PRD.md#75-小程序练习页核心页面)

```
┌────────────────────────────────┐
│  ——                            │
│  答题卡                    ✕   │
├────────────────────────────────┤
│  ■ 已做    □ 未做              │  图例
│                                │
│  ┌──┬──┬──┬──┬──┬──┬──┐     │
│  │1 │2 │3 │4 │5 │6 │7 │     │  7 列网格
│  ├──┼──┼──┼──┼──┼──┼──┤     │  已做=实心
│  │8 │...                     │  未做=空心
│  └──┴──┴──┴──┴──┴──┴──┘     │  当前题=加粗边框
│                                │
└────────────────────────────────┘
```

| 属性 | 说明 |
|------|------|
| 弹出方式 | 从底部滑入，半屏 |
| 图例 | 已做 / 未做 |
| 当前题高亮 | 加粗边框 |
| 点击题号 | 跳转到对应题目，关闭弹窗 |
| 关闭方式 | 点击 ✕ 或点击遮罩 |

### 4.3 提前结束确认弹窗

> **详见** [整体PRD §7.5 提前结束确认弹窗](./练习功能整体PRD.md#75-小程序练习页核心页面)

```
┌─────────────────────┐
│                     │
│       提示          │
│ 你还有未完成试题，    │
│ 确定要交卷吗？      │
│                     │
├─────────────────────┤
│       确定          │
├─────────────────────┤
│       取消          │
└─────────────────────┘
```

- **确定** → 跳转「练习结束页面」
- **取消** → 关闭弹窗，继续答题

## 5. 核心交互

| 操作 | 行为 |
|------|------|
| 进入页面 | 后台按练习配置从试卷抽题，渲染首题 |
| 选择答案 | 记录选项，高亮选中项 |
| 上一题 | 回到前一题，可修改已选答案；首题时按钮置灰不可用 |
| 下一题 | 保存当前答案，前进到下一题 |
| 最后一题 → 提交 | 按钮文案变为「提交」，点击后跳转练习结束页 |
| 返回 | 弹出确认对话框（标题：确认退出练习，内容：退出后将不会保存本次练习，请谨慎操作，按钮：取消/退出），点击「退出」返回练习列表 |
| 提前结束 | 弹出确认弹窗，确定后跳转练习结束页 |

### 5.1 结束条件

> **详见** [整体PRD §7.5 结束条件](./练习功能整体PRD.md#75-小程序练习页核心页面) 及 [整体PRD §8.2 用户练习生命周期](./练习功能整体PRD.md#82-用户练习生命周期)

| 结束方式 | 触发条件 | 行为 |
|---------|---------|------|
| 做完提交 | 答完最后一题 → 点击「提交」 | 跳转练习结束页 |
| 提前结束 | 点击「提前结束」→ 确认弹窗 → 确定 | 跳转练习结束页 |
| 返回退出 | 点击导航栏返回 → 确认弹窗 → 退出 | 返回练习列表，不保存进度 |

## 7. 与自由练习页面的关系

| 对比项 | 自由练习 | 模拟练习 |
|--------|---------|---------|
| 页面结构 | 相同 | 复用 |
| 页面样式 | 现有样式 | 完全复用 |
| **导航栏标题** | **自由练习标题** | **固定显示「模拟练习」** |
| 按钮功能 | 答题卡、提前结束、上一题、下一题/提交 | 全部不变 |
| 题目来源 | 通用题库 + 专业题库随机抽题 | 关联试卷按配置抽题 |
| 答题卡弹窗 | 相同 | 复用 |
| 提前结束弹窗 | 相同 | 复用 |
| 结束页面跳转 | 自由练习结束页 | 模拟练习结束页（小程序练习结束页面.html） |

## 8. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 题目数据加载失败 | 轻提示「加载失败，请稍后重试」 |
| 练习已结束（过期访问） | 提示「练习已结束」，返回列表页 |
| 网络中断 | 保留当前已作答记录，网络恢复后可继续 |
| 关联试卷已删除 | 不影响练习，已抽取的题目数据可正常使用 |
| 用户被移除练习人员范围 | 不影响已进入的练习，可继续答题至结束 |
