# 小程序考试中心 · 页面优化 PRD

> **页面类型**：用户前台类（小程序端）
> **所属模块**：考试中心
> **最后更新**：2026-07-02

---

## 1. 引言

### 1.1 变更概述

小程序考试中心为**已有页面**，是员工查看考试任务与进入练习的入口页。用户通过工作台「考试中心」入口进入，页面提供快捷入口（学习积分、自由练习、模拟练习、排行榜）及考试列表（进行中/已结束）。


### 1.2 使用角色

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 可见快捷入口与考试任务 |
| 部门负责人 | ✓ 可见快捷入口与考试任务 |
| 超级管理员 | ✗（使用管理后台） |
| 安全管理员 | ✗（使用管理后台） |

---

## 2. 变更内容

### 2.1 快捷入口（变更区域）

考试中心顶部快捷入口区域，原有「自由练习」「学习积分」「排行榜」三个入口图标。

**本次变更**：
- 在「学习积分」之后新增「模拟练习」图标

**变更后快捷入口**（共 4 个）：

| # | 入口名称 | 图标 | 点击行为 | 状态 |
|:--:|---------|------|---------|:----:|
| 1 | 学习积分 | 积分数字 | 查看学习积分 | 已有 |
| 2 | 自由练习 | 时钟图标 | 跳转自由练习刷题页 | 已有 |
| 3 | **模拟练习** | 试卷图标 | **跳转小程序练习列表** | **新增** |
| 4 | 排行榜 | 📊 图标 | 查看排行榜 | 已有 |

### 2.2 模拟练习入口详情

| 属性 | 值 |
|------|----|
| 位置 | 快捷入口区第 3 项，「自由练习」之后 |
| 图标 | 📝 模拟练习 |
| 点击行为 | 跳转至 `小程序练习列表.html`（模拟练习列表页） |

### 2.3 模拟练习入口可见性

模拟练习入口**不受资源授权和练习权限影响**，入口始终对所有用户可见。

---

## 3. 交互流程

```
用户点击考试中心「模拟练习」入口
         │
         ▼
跳转至模拟练习列表页
         │
         ├── 有数据 → 展示练习卡片列表
         └── 无数据 → 展示空状态
```

---

## 4. 与已有功能的关系

| 已有组件 | 影响 |
|----------|------|
| 考试列表（进行中/已结束） | 无影响 |
| 用户信息区 | 无影响 |
| 搜索筛选栏 | 无影响 |
| 导航栏 | 无影响 |
| 其他快捷入口（自由练习、学习积分、排行榜） | 无影响，继续正常使用 |

## 5. 数据来源

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 快捷入口配置 | 页面固定配置 | 模拟练习入口始终可见，不受菜单资源授权控制 |
| 模拟练习列表数据 | 后端接口 | 按用户权限返回已发布练习 |
