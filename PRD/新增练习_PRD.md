# 新增练习 PRD（产品需求文档）

## 1. 引言

### 1.1 背景

新增练习页面用于管理员创建一条模拟练习草稿。管理员选择试卷、配置题目数量、设置练习开放时间，并指定参与练习的部门、岗位或人员。保存后练习保持草稿状态，发布操作统一回到练习列表完成。

### 1.2 页面入口

| 来源 | 行为 |
|------|------|
| 练习列表点击「新增」 | 跳转至 `练习设置_新增模拟练习.html` |
| index 导航点击「新增练习」 | 打开新增练习页面 |

### 1.3 使用角色

- **管理员**：可创建练习草稿。
- **平台员工**：无此页面访问权限。

## 2. 页面布局

新增练习为独立页面，包含 4 个表单分组。

```text
+-------------------------------------------------------+
|  ← 新增模拟练习                                       |
+-------------------------------------------------------+
|  基本信息                                             |
|    练习名称、练习开放时间                              |
|                                                       |
|  选择试卷                                             |
|    选择试卷、试卷结构信息                              |
|                                                       |
|  练习题目数量                                         |
|    练习题目总数、单选/多选/判断数量                    |
|                                                       |
|  练习人员                                             |
|    添加人员、部门/岗位/人员已选标签、查看更多抽屉       |
+-------------------------------------------------------+
|                 [返回] [保存]                         |
+-------------------------------------------------------+
```

## 3. 表单字段

| 字段 | 控件类型 | 必填 | 默认值 | 校验规则 | 交互说明 |
|------|----------|:----:|--------|----------|----------|
| 练习名称 | 文本输入框 | 是 | 空 | ≤20 字 | 实时显示字数计数 |
| 练习开放时间 | 日期时间范围 | 是 | 空 | 结束时间 > 开始时间；开始时间 > 当前时间 | 员工可在开放时间内练习 |
| 选择试卷 | 只读输入框 + 试卷选择弹窗 | 是 | 空 | 必选 | 单选试卷，选择后展示试卷结构 |
| 练习题目总数 | 数字输入框 | 是 | 空 | >0 且 ≤ 试卷题目总数 | 未选择试卷前禁用 |
| 各题型数量 | 复选框 + 数字输入 | 是 | 空 | 至少选择一种题型；题型合计 = 总数；各题型不超过试卷上限 | 选择试卷后启用 |
| 练习人员 | 人员选择器 | 是 | 空 | 至少选择一个部门、岗位或人员 | 支持部门、岗位、人员多选 |

## 4. 选择试卷

### 4.1 打开方式

- 点击「选择试卷」输入框或选择按钮打开试卷选择弹窗。
- 试卷为单选。

### 4.2 选择后展示

选择试卷后，在表单中展示试卷结构信息：

| 信息 | 说明 |
|------|------|
| 试卷名称 | 当前选择的试卷名称 |
| 题目总数 | 试卷内题目总量 |
| 单选题数量 | 试卷内单选题数量 |
| 多选题数量 | 试卷内多选题数量 |
| 判断题数量 | 试卷内判断题数量 |

### 4.3 题目数量联动

- 未选择试卷前，题目数量相关控件禁用。
- 选择试卷后，题目数量控件启用。
- 各题型数量不能超过试卷对应题型数量。

## 5. 练习人员

### 5.1 人员选择器

点击「添加人员」打开人员选择弹窗。

| 页签 | 支持能力 |
|------|----------|
| 部门 | 支持按部门名称查询、勾选多个部门 |
| 岗位 | 支持按岗位名称查询、勾选多个岗位 |
| 人员 | 支持按姓名、部门查询，勾选多个人员 |

### 5.2 人员选择器分页

- 部门、岗位、人员列表均支持分页。
- 当列表数据超过 10 条时展示分页。
- 默认每页 10 条。
- 翻页不清空已选数据。

### 5.3 已选内容展示

选择完成后，在表单中按类型分组展示：

| 类型 | 展示标题 | 展示规则 |
|------|----------|----------|
| 部门 | 部门（X个） | 最多直接展示 10 项 |
| 岗位 | 岗位（X个） | 最多直接展示 10 项 |
| 人员 | 人员（X个） | 最多直接展示 10 项 |

- 每个标签右侧展示删除图标，支持逐项移除。
- 已选内容在表单内直接分组展示，超过直接展示上限时通过「查看更多X项」进入抽屉查看完整列表。
- 当某类型已选数量超过 10 项时，在第 10 项后展示「查看更多X项」按钮。
- X 为剩余未直接展示的数量。

### 5.4 查看更多抽屉

点击「查看更多X项」后打开对应类型的已选列表抽屉。

| 抽屉 | 标题 | 查询条件 | 表格字段 | 操作 |
|------|------|----------|----------|------|
| 已选部门列表 | 已选部门列表 | 部门名称 | 序号、部门名称、操作 | 删除、批量删除 |
| 已选岗位列表 | 已选岗位列表 | 岗位名称 | 序号、岗位名称、操作 | 删除、批量删除 |
| 已选人员列表 | 已选人员列表 | 姓名、部门名称 | 序号、姓名、部门、操作 | 删除、批量删除 |

抽屉规则：

- 抽屉从右侧打开，宽度与编辑页面保持一致。
- 支持查询、重置。
- 支持勾选后批量删除。
- 未勾选时点击批量删除，提示「请选择要批量操作的内容」。
- 点击「返回」关闭抽屉并回到新增表单。

## 6. 底部按钮

| 按钮 | 行为 |
|------|------|
| 返回 | 返回练习列表，不保存当前填写内容 |
| 保存 | 执行表单校验，校验通过后保存为草稿 |

保存成功后提示：

> 练习已保存成功，您可以通过点击「发布」按钮完成发布

## 7. 表单校验

校验不通过时展示提示，不离开页面。

1. 练习名称必填。提示：「请输入练习名称」。
2. 请选择试卷。提示：「请选择试卷」。
3. 练习开放时间必须完整填写。提示：「请填写完整的练习开放时间」。
4. 结束时间必须大于开始时间。提示：「结束时间必须大于开始时间」。
5. 开始时间必须大于当前时间。提示：「开始时间必须大于当前时间」。
6. 题目总数必须大于 0。提示：「请输入有效的题目总数」。
7. 题目总数不能超过试卷题目总数。提示：「题目总数不能超过试卷题目总数（N 道）」。
8. 至少选择一种题目类型。提示：「请至少选择一种题目类型」。
9. 各题型数量之和必须等于题目总数。提示：「各题型数量之和需等于题目总数」。
10. 各题型数量不能超过试卷对应题型数量。提示：「不能超过试卷单选/多选/判断题数量（N 道）」。
11. 至少选择一个部门、岗位或人员。提示：「请至少添加一个人员或部门」。

## 8. 保存数据

| 字段 | 说明 |
|------|------|
| 练习名称 | 管理员输入的练习名称 |
| 试卷信息 | 选中的试卷 ID、名称、题目结构 |
| 题目数量 | 题目总数及各题型数量 |
| 开放时间 | 开始时间、结束时间 |
| 练习对象 | 已选部门、岗位、人员 |
| 人员快照 | 根据练习对象计算出的当前应练人员 |
| 发布状态 | 保存后默认为草稿 |
