## 1. 引言

### 1.1 页面概述

> **详见** [整体PRD §7.6 小程序练习结束页面](./练习功能整体PRD.md#76-小程序练习结束页面)

练习结束页面是模拟练习答题完成后的结果展示页。页面展示本次练习的得分情况、正确率圆环、答题统计，以及逐题答题卡。用户可查看成绩详情或再次练习。**本页面复用自由练习的练习结束页面，样式与自由练习完全一致。**

### 1.2 使用角色

> **详见** [整体PRD §2 角色与权限](./练习功能整体PRD.md#2-角色与权限)

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 从练习页面提交或提前结束后进入 |
| 部门负责人 | ✓ 从练习页面提交或提前结束后进入 |

### 1.3 前置条件

- 从练习页面完成答题（提交或提前结束）后跳转进入
- 携带本次练习结果数据

## 2. 页面结构

### 2.1 页面布局

```
┌──────────────────────────────┐
│       ··· (状态栏)            │
├──────────────────────────────┤
│  ←  考试结果                   │  导航栏
├──────────────────────────────┤
│                              │
│       XXXX练习                │  练习名称
│                              │
│       ◯◯◯◯◯◯◯               │  圆环进度
│       ◯  60% ◯               │  正确率
│       ◯◯◯◯◯◯◯               │
│                              │
│   100           60           │  答题总数 / 答对数量
│                              │
│  接近全对，再努力一点          │  鼓励语
│                              │
├──────────────────────────────┤
│  答题卡    ■答对 ■答错 ■未做  │
│                              │
│  ┌──┬──┬──┬──┬──┐            │
│  │ 1│ 2│ 3│ 4│ 5│           │  5 列网格
│  ├──┼──┼──┼──┼──┤           │  答对=绿色
│  │ 6│...                    │  答错=红色
│  └──┴──┴──┴──┴──┘           │  未做=灰色
│                              │
├──────────────────────────────┤
│  [查看详情]       [再次练习]   │  底部操作栏（固定）
└──────────────────────────────┘
```

### 2.2 导航栏

| 元素 | 位置 | 行为 |
|------|------|------|
| 返回按钮 ← | 左侧 | 返回练习列表页 |
| 标题 | 居中 | 固定显示「考试结果」 |

## 3. 结果头部

### 3.1 得分展示

| 元素 | 说明 |
|------|------|
| 练习名称 | 显示本次练习的名称 |
| 圆环进度 | 环形图表，按正确率百分比填充 |
| 正确率数字 | 圆环中央显示百分比数字 + % 符号 |
| 正确率标签 | 圆环中央下方显示「正确率」文字 |

**圆环颜色规则：**
- 高分段：正确率 100%
- 中间段：正确率 60%~99%
- 低分段：正确率低于 60%

### 3.2 统计数字

| 指标 | 位置 | 说明 |
|------|------|------|
| 答题总数 | 左侧 | 本次练习题目总数 |
| 答对数量 | 右侧 | 答对的题目数 |

### 3.3 鼓励语

> **详见** [整体PRD §7.6 鼓励语3档规则](./练习功能整体PRD.md#76-小程序练习结束页面)

| 正确率 | 文案 |
|--------|------|
| 100% | 全对！你太棒了，继续保持！ |
| 60%~99% | 接近全对，再努力一点，下次完美！ |
| < 60% | 别灰心，总结经验，反复练习！ |

## 4. 答题卡

### 4.1 区域结构

| 元素 | 说明 |
|------|------|
| 标题 | 左侧「答题卡」 |
| 图例 | 右侧横向排列：答对 / 答错 / 未做，各带数量和颜色方块 |
| 题号网格 | 5 列等宽网格，每格显示题号数字 |

### 4.2 题号状态

| 状态 | 说明 |
|------|------|
| 答对 | 绿色背景，显示题号 |
| 答错 | 红色背景，显示题号 |
| 未做 | 灰色背景，显示题号 |

### 4.3 图例数据

| 图例项 | 数据来源 |
|--------|----------|
| 答对 | 答对题目数量 |
| 答错 | 答错题目数量 |
| 未做 | 未作答题目数量 |

## 5. 底部操作栏

### 5.1 按钮

| 按钮 | 位置 | 行为 |
|------|:--:|------|
| 查看详情 | 左侧 | 跳转试卷详情页面，逐题回顾答题情况 |
| 再次练习 | 右侧 | 弹出确认弹窗，确定后重新进入练习页 |

### 5.2 再次练习确认弹窗

> **详见** [整体PRD §7.6 再次练习确认弹窗](./练习功能整体PRD.md#76-小程序练习结束页面)

```
┌─────────────────────┐
│                     │
│       提示          │
│  确定重新练习吗？    │
│                     │
├─────────────────────┤
│       确定          │
├─────────────────────┤
│       取消          │
└─────────────────────┘
```

- **确定** → 跳转练习页面，重新开始练习
- **取消** → 关闭弹窗，停留在当前页面

## 6. 核心交互

| 操作 | 行为 |
|------|------|
| 进入页面 | 接收练习结果数据，渲染得分、统计、答题卡和鼓励语 |
| 返回 | 跳转到练习列表页 |
| 查看详情 | 跳转试卷详情页面，逐题查看答案与解析 |
| 再次练习 | 弹出确认弹窗 → 确定 → 跳转练习页面开始新练习 |
| 再次练习 → 取消 | 关闭弹窗，留在当前页 |

## 7. 数据来源

### 7.1 结果数据

| 字段 | 说明 |
|------|------|
| 练习名称 | 本次练习的名称 |
| 答题总数 | 本次练习题目总数量 |
| 答对数量 | 答对的题目数 |
| 答错数量 | 答错的题目数 |
| 未做数量 | 未作答的题目数 |
| 正确率 | 答对数量 / 答题总数 × 100% |
| 每题状态 | 每题答对 / 答错 / 未做的状态标识 |

## 8. 与自由练习页面的关系

| 对比项 | 自由练习 | 模拟练习 |
|--------|---------|---------|
| 页面结构 | 相同 | 复用 |
| 页面样式 | 现有样式 | 完全复用 |
| **顶部名称字段** | **专业名称** | **改为练习名称（如「模拟练习」）** |
| 按钮功能 | 查看详情、再次练习 | 全部不变 |
| 题目数据来源 | 自由练习答题记录 | 模拟练习答题记录 |
| 查看详情跳转 | 自由练习试卷详情页 | 模拟练习试卷详情页（小程序试卷详情页面.html） |

---

## 9. 页面关系

| 来源页面 | 跳转方式 |
|----------|----------|
| 练习页面（提交） | 答题完成后点击「提交」跳转 |
| 练习页面（提前结束） | 点击「提前结束」→ 确认 → 跳转 |

| 目标页面 | 触发操作 |
|----------|----------|
| 练习列表页 | 点击导航栏 ← 返回 |
| 试卷详情页面 | 点击「查看详情」 |
| 练习页面 | 点击「再次练习」→ 确认 |

---

## 10. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 结果数据缺失 | 显示默认空状态，提示「暂无结果数据」 |
| 全部未作答提交 | 正确率按 0% 显示，鼓励语对应低分段 |
| 跳转目标页面不存在 | 轻提示「页面加载失败」，停留在当前页 |
