## 1. 引言

### 1.1 页面概述

> **详见** [整体PRD §7.3 小程序工作台-新增入口](./练习功能整体PRD.md#73-小程序工作台---新增入口)

小程序工作台是员工进入小程序后的首页，以图标网格形式展示各功能入口。本次在「学习管理」区块新增**模拟练习**入口图标，供用户进入模拟练习列表。

### 1.2 使用角色

> **详见** [整体PRD §2 角色与权限](./练习功能整体PRD.md#2-角色与权限)

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 可见且有权限的功能入口 |
| 部门负责人 | ✓ 可见且有权限的功能入口 |
| 超级管理员 | ✗（使用管理后台） |
| 安全管理员 | ✗（使用管理后台） |

### 1.3 前置条件

- 管理后台「菜单资源授权」已开启练习功能模块的授权
- 小程序工作台已渲染「学习管理」区块及相关图标

## 4. 权限控制

### 4.1 菜单资源授权

图标可见性由已有的**菜单资源授权**功能统一控制：

| 授权状态 | 效果 |
|---------|------|
| 已授权「模拟练习」菜单 | 工作台显示「模拟练习」图标 |
| 未授权「模拟练习」菜单 | 工作台不显示「模拟练习」图标 |

### 4.2 图标展示规则

- 管理员在后台「菜单资源授权」中为角色/用户授权后，对应图标出现在工作台
- 未授权的图标不渲染，不影响其他图标布局
- 图标数量变化时，网格自动调整（4列固定，行数随图标数变化）

| 点击「模拟练习」图标 | 跳转至 `小程序练习列表.html` |

| 模块 | 关系 | 说明 |
|------|------|------|
| 菜单资源授权 | 依赖 | 已有功能，控制工作台图标可见性。本次「模拟练习」图标复用该授权机制 |
