# 小程序试卷详情页面 · 页面 PRD

> **页面类型**：用户前台类（小程序端）
> **所属模块**：模拟练习
> **最后更新**：2026-07-01

---

## 1. 引言

### 1.1 页面概述

小程序试卷详情页面从练习结束页点击「查看详情」进入，逐题展示正确答案、用户答案和题目解析。**本页面复用自由练习的试卷详情页面，样式与自由练习完全一致**，题目来源改为模拟练习的答题记录，所有按钮功能保持不变。

### 1.2 使用角色

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 从练习结束页进入 |
| 部门负责人 | ✓ 从练习结束页进入 |

### 1.3 前置条件

- 用户已完成一轮模拟练习并提交（或提前结束）
- 从练习结束页点击「查看详情」进入

---

├──────────────────────────────┤
│                              │
│  ┌──────────────────────────┐│
│  │ 1. 单选题      ✓答对     ││  题号 + 题型 + 状态
│  │                          ││
│  │ 安全生产第一责任人是？     ││  题目内容
│  │                          ││

### 2.2 导航栏

| 元素 | 位置 | 行为 |
|------|------|------|
| 返回按钮 ← | 左侧 | 点击返回练习结束页面 |
| 标题「试卷详情」 | 居中 | 纯文本展示 |

---

## 3. 进度模块

| 元素 | 说明 |
|------|------|
| 标签 | 「题目进度」 |
| 计数器 | 当前题号 / 总题数，格式 `1 / 4` |
| 进度条 | 填充条，按 (当前题/总题数) 比例 |

---

## 4. 题目卡片

### 4.1 卡片元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 题号 | 左上 | 题号 + 题型标签（单选/多选/判断） |
| 状态标签 | 右上 | ✓答对 / ✗答错 |
| 题目内容 | 卡片中部 | 题目文字 |
| 选项列表 | 题目下方 | 正确答案标记、错选标记、未选标记 |
| 正确答案 | 解析区上方 | 「正确答案」+ 答案字母 |
| 我的答案 | 正确答案下方 | 「我的答案」+ 答案字母（或「未作答」） |
| 解析 | 解析区 | 题目解析文字 |

### 4.2 选项状态

| 状态 | 图标 |
|------|:--:|
| 正确答案 + 用户答对 | ✓ |
| 正确答案 + 用户未选 | ✓ |
| 用户错选 | ✗ |
| 其他未选选项 | - |

### 4.3 分页逻辑

一题一页，通过「下一题」按钮逐题切换。切换时实时更新：
- 进度条填充比例
- 计数器数字
- 题目卡片内容

---

## 5. 底部操作栏

### 5.1 按钮

| 按钮 | 位置 | 行为 |
|------|:--:|------|
| 返回首页 | 左侧 | 跳转练习列表页 |
| 上一题 | 中间 | 回到上一题，查看前一题的答题详情 |
| 下一题 | 右侧 | 切换到下一题；最后一题变为「返回结果」 |

### 5.2 下一题 / 返回结果按钮状态

| 当前题 | 按钮状态 |
|--------|---------|
| 不是最后一题 | 正常可点击，文案「下一题」 |
| 最后一题 | 文案变为「返回结果」，点击跳转到练习结束页面 |

### 5.3 上一题按钮状态

| 当前题 | 按钮状态 |
|--------|---------|
| 第一题（题号 1） | 隐藏不显示 |
| 第二题起（题号 ≥ 2） | 显示「上一题」，正常可点击 |

---

## 6. 核心交互

| 操作 | 行为 |
|------|------|
| 进入页面 | 渲染第一道题目详情，不显示「上一题」按钮 |
| 点击「下一题」 | 切换到下一题，更新进度条和计数器；切换到第 2 题后显示「上一题」 |
| 点击「上一题」 | 切换回上一题，更新进度条和计数器 |
| 点击「返回结果」 | 最后一题按钮，点击跳转到练习结束页面 |
| 点击「返回首页」 | 跳转练习列表页 |
| 点击返回按钮 | 返回练习结束页面 |

---

## 7. 数据来源

### 7.1 题目数据

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 题目列表 | 本轮练习的答题记录 | 包含题目、用户答案、正确答案、解析 |
| 答对/答错状态 | 答题记录 | 对比用户答案与正确答案判定 |
| 题目进度计数 | 答题记录 | 总题数 = 本次练习题目总数 |

> **试卷删除兼容**：题目数据在用户进入练习时已从试卷中抽取并落库存储，原试卷删除不影响已完成的练习结果数据，用户仍可正常查看试卷详情。

### 7.2 与自由练习试卷详情页面的关系

| 对比项 | 自由练习 | 模拟练习 |
|--------|---------|---------|
| 页面结构 | 相同 | 复用 |
| 页面样式 | 现有样式 | 完全复用 |
| **上一题按钮** | 第 2 题起显示 | 不变 |
| **最后一题按钮** | 最后一题变为「返回结果」 | 不变 |
| 返回首页按钮 | 返回首页 | 不变 |
| 题目数据来源 | 自由练习答题记录 | 模拟练习答题记录 |
| 返回按钮目标 | 自由练习结束页 | 模拟练习结束页（小程序练习结束页面.html） |
| 返回首页目标 | 练习列表页 | 练习列表页（相同） |

---

## 8. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 无答题记录数据 | 展示空状态「暂无答题详情」，返回首页按钮可用 |
| 数据加载失败 | 轻提示「加载失败，请稍后重试」 |
