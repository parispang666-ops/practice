## 1. 引言

### 1.1 页面概述

> **详见** [整体PRD §7.4 小程序模拟练习列表](./练习功能整体PRD.md#74-小程序模拟练习列表)

小程序练习列表页面是员工在小程序端进入「模拟练习」模式时的中间页，用于展示当前用户有权限参与的所有模拟练习卡片，供用户选择并进入具体练习。

### 1.2 使用角色

> **详见** [整体PRD §2 角色与权限](./练习功能整体PRD.md#2-角色与权限)

| 角色 | 访问权限 |
|------|----------|
| 普通员工 | ✓ 可见且有权限的练习 |
| 部门负责人 | ✓ 可见且有权限的练习 |
| 超级管理员 | ✗（使用管理后台） |
| 安全管理员 | ✗（使用管理后台） |

### 1.3 前置条件

- 管理后台已开启练习功能的资源授权
- 小程序工作台已集成「练习」入口图标
- 满足以上两点即可进入练习列表（无有效练习时展示空数据）

## 2. 页面路由逻辑

页面展示由有效模拟练习数量决定：

| 用户可见试卷数 | 展示结果 |
|:------------:|----------|
| 0 条 | 展示**列表空数据**（提示「暂无数据」） |
| ≥1 条 | 展示**模拟练习列表** |

> **用户可见试卷数** = 已发布 + 当前用户在练习人员范围内 的试卷数量。
>
> 「练习人员范围」判定逻辑：
> - 用户在该练习的「练习人员」列表中：可见
> - 用户不在该练习的「练习人员」列表中：不可见

### 2.2 发布后增删人员规则

> **详见** [整体PRD §7.4 发布后增删人员规则](./练习功能整体PRD.md#74-小程序模拟练习列表)

当管理员在练习发布后增删练习人员时：

| 操作 | 已有人员（已进入练习中） | 已有人员（未进入练习） | 新增人员 |
|------|------------------------|----------------------|----------|
| 列表展示 | 刷新后不展示 | 刷新后不展示 | 列表展示 |
| 能否练习 | 可继续当前练习直到结束 | 无法进入练习 | 可正常练习 |
| 查看试卷详情 | 可从练习结束页查看 | — | 可查看 |

> **说明**：
> - **已进入练习中的人员被移除后**：不中断当前练习，可继续答题至结束、查看试卷详情；但刷新页面后列表不再展示该练习卡片，无入口重新进入。
> - **新增人员**：发布后新加入的人员，列表正常展示该练习卡片，可正常进入练习。

## 4. 试卷卡片

### 4.1 卡片元素

| 元素 | 位置 | 说明 |
|------|------|------|
| 练习名称 | 卡片左上 | 管理后台配置的名称，单行截断 |
| 状态标签 | 卡片右上 | 三种状态：未开始 / 进行中 / 已结束 |
| 开放时间 | 名称下方 | 显示「开放时间」+ 开始时间 ~ 结束时间 |
| 前往练习按钮 | 卡片底部 | **仅「进行中」状态显示** |

### 4.2 状态标签规则

> **详见** [整体PRD §7.4 状态说明](./练习功能整体PRD.md#74-小程序模拟练习列表) 及 [整体PRD §4 名词解释](./练习功能整体PRD.md#4-名词解释)

| 状态 | 判定条件 | 标签文案 |
|------|----------|:--------:|
| 未开始 | 当前时间早于练习开始时间 | 未开始 |
| 进行中 | 当前时间在练习开始时间和结束时间之间，或无结束时间 | 进行中 |
| 已结束 | 当前时间晚于练习结束时间 | 已结束 |

> 若未设置开放时间，状态显示为「无」。

### 4.3 卡片排序

按**创建时间倒序**排列（最新创建的练习排在最前）。

### 4.4 卡片交互

| 交互操作 | 行为 |
|----------|------|
| 点击「进行中」卡片（任意位置） | 跳转该模拟练习页面 |
| 点击「未开始」卡片 | 不响应跳转 |
| 点击「已结束」卡片 | 不响应跳转 |

## 6. 数据来源

### 6.1 后台配置字段

数据来源于管理后台「练习设置 → 模拟练习」中已保存的练习配置：

| 字段 | 类型 | 说明 |
|------|------|------|
| 练习编号 | 文本 | 练习唯一标识 |
| 练习名称 | 文本 | 管理后台配置的练习名称 |
| 开放开始时间 | 日期时间 | 练习开放起始时间 |
| 开放结束时间 | 日期时间 | 练习开放截止时间 |
| 创建时间 | 日期时间 | 用于排序，最新创建排前 |
| 是否已发布 | 是/否 | 标记是否已发布 |
| 练习人员 | 人员列表 | 指定可参与练习的人员 |

## 7. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 无有效模拟练习 | 展示列表空状态：「暂无数据」 |
| 接口请求失败 | 轻提示「加载失败，请稍后重试」（不展示空状态，保留上次数据） |
| 用户无权限（被排除在人员范围外） | 对应试卷不展示在列表中，逻辑等同于无数据 |
