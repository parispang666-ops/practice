# 环慧慧 EHS+ 练习功能 · 原型预览

HWW 练习功能模块的 HTML 原型页面集合，包含 PC 后台管理端和小程序端页面。

## 预览入口

[查看页面导航](https://parispang666-ops.github.io/practice/)

## 页面清单

### 电脑端（PC 后台）
- **练习列表** — 练习列表页面（含专业练习 + 模拟练习 Tab）
- **新增练习** — 新增练习页面
- **编辑练习** — 编辑练习页面
- **练习记录** — 练习记录页面

### 小程序端
- **工作台** — 新增「模拟练习」入口卡片
- **练习列表** — 已发布模拟练习卡片列表
- **练习页面** — 逐题作答 + 答题卡弹窗
- **练习结束** — 练习结果统计 + 回顾
- **试卷详情** — 逐题查看答案与解析
- **考试中心** — 考试入口与成绩查看
