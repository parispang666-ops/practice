练习设置 - 模拟练习 PRD（产品需求文档）

1. 引言

1.1 背景

在练习设置页新增「模拟练习」Tab，管理员选择关联标准试卷，按试卷结构配置练习参数（题目总数、各题型数量、练习开放时间、练习人员）。模拟练习支持灵活划定题目范围，进行针对性强化训练，适用于考前冲刺与阶段性能力评估。员工在小程序端按配置进行刷题练习。本需求仅覆盖模拟练习相关功能，不涉及已存在的「专业练习」Tab。

1.2 使用角色

  • 管理员：可查看、新增、编辑、删除、发布模拟练习配置
  • 平台员工：无此页面访问权限（仅在小程序端参与练习）

1.3 前置条件

  • 用户已登录系统
  • 用户具有「练习设置」菜单权限（管理员角色）
  • 系统中已存在试卷数据（用于选择器）

注意：试卷仅用于创建练习时的题目配置来源，练习创建后与试卷解耦。试卷的删除不影响已创建的练习功能，已发布的练习仍可正常使用。

1.4 功能清单

模块功能功能描述
列表区模拟练习列表以表格形式展示所有模拟练习配置（名称、时间、状态、创建人、创建时间），支持按创建时间倒序排列
操作区行操作根据状态显示不同操作：草稿→发布+编辑+删除；未开始→编辑；进行中→编辑；已结束→-
弹窗新增练习弹窗表单（4个分组），录入练习配置，支持保存草稿或保存并发布
弹窗编辑练习回填已有数据。草稿/未开始：全部字段可编辑，底部「返回+保存+保存并发布」（草稿）或「返回+保存设置」（未开始，点击保存设置二次确认后保存）；进行中：仅可编辑部分字段，底部「返回+保存设置」（点击保存设置二次确认后保存）

3. 需求说明

3.1 列表区

列表字段：

列表字段字段说明排序
练习名称管理员自定义名称，≤20 字，单行展示否
开始时间练习开放开始时间（年-月-日 时:分）否
结束时间练习开放结束时间（年-月-日 时:分）否
状态草稿/未开始/进行中/已结束否
创建人创建该练习的管理员姓名否
创建时间创建时间（年-月-日 时:分:秒）是（倒序）

排序规则：按创建时间倒序，最新创建的排在前面。

空状态：列表无数据时显示「暂无数据」。

3.2 行操作说明

根据练习状态，每行末尾显示不同操作按钮：

状态操作按钮说明
草稿发布 + 编辑 + 删除可发布、编辑、删除
未开始编辑仅可编辑
进行中编辑仅可编辑
已结束-不可操作

3.2.1 发布

  • 显示条件：状态为「草稿」
  • 行为：发布前先校验练习开放时间：须完整填写、结束>开始、开始>当前时间。校验通过后将是否已发布设为已发布，刷新列表
  • 状态变化：草稿 → 未开始/进行中/已结束（由开放时间与当前时间关系自动判定）
  • 反馈：提示「发布成功」，3 秒后自动消失

发布前校验（按顺序）：

  1. 练习开放时间必须完整填写 →「请先编辑练习，填写完整的练习开放时间后再发布」
  2. 结束时间必须大于开始时间 →「结束时间必须大于开始时间，请先修改后再发布」
  3. 开始时间必须大于当前时间 →「开始时间必须大于当前时间，请先修改后再发布」

3.2.2 编辑

  • 显示条件：草稿、未开始、进行中（已结束不显示）
  • 行为：打开右侧表单页，回填全部已有数据
    • 草稿状态：全部字段可编辑，试卷可选
    • 未开始状态：全部字段可编辑，试卷可选可改
    • 进行中状态：试卷选择器、练习名称、题目数量、开始时间置灰不可修改，结束时间仅可延长（下限取当前时间和开始时间的较大值），可编辑练习人员

3.2.3 删除

  • 显示条件：草稿
  • 行为：弹出确认弹窗，文案「确定要删除该练习吗？此操作不可恢复。」
  • 确认后：从数据列表移除，刷新表格
  • 反馈：提示「删除成功」，3 秒后自动消失
  • 禁用情况：进行中状态下提示「进行中的练习不可删除」；已结束状态不显示删除入口

3.3 新增/编辑弹窗

弹窗以右侧表单页形式展示（替代浮层弹窗），左侧列表隐藏，右侧显示完整表单页，左上角有返回箭头按钮。

表单分组（4 个分组，从上到下依次为）：

分组说明
基本信息练习名称 + 练习开放时间
选择试卷试卷选择器 + 试卷结构信息条
练习题目数量题目总数 + 各题型数量，未选择试卷前全部禁用
练习人员人员选择器

新增模式 vs 编辑模式：

对比项新增模式编辑模式
标题「新增模拟练习」「编辑模拟练习」
练习名称可编辑草稿/未开始可编辑；进行中置灰禁用
试卷选择可点击打开选择器草稿/未开始可编辑；进行中置灰禁用，不可点击
题目数量选试卷后可编辑草稿/未开始可编辑；进行中置灰禁用
开始时间可编辑进行中置灰禁用；草稿/未开始可编辑
结束时间可编辑进行中仅可延长（下限取当前时间和开始时间的较大值）；草稿/未开始可自由编辑
初始数据全部默认值（空）回填已有数据

3.3.1 表单字段

字段标签控件类型必填默认值（新增）校验规则交互说明
练习名称文本输入框✓空≤20 字实时显示字数计数 0/20
练习开放时间两个日期时间输入框✓空结束>开始；新增时开始>当前时间格式 年-月-日 时:分；开始时间 至 结束时间；右侧提示「员工可以在练习开放时间内进行练习」

3.3.2 底部按钮

新增模式 / 草稿编辑模式：

按钮行为
返回返回左侧列表，不保存任何修改
保存校验通过后保存，状态为未发布，列表显示「草稿」；提示「新增成功」/「修改成功」
保存并发布校验通过后保存，状态为已发布，列表根据时间范围显示对应状态；提示「新增成功」/「修改成功」

未开始 / 进行中编辑模式：

按钮行为
返回返回左侧列表，不保存任何修改
保存设置点击弹出确认弹窗二次确认「确定要保存当前设置吗？」，确认后校验通过则保存，提示「修改成功」

编辑模式保存规则：「保存」保持原发布状态不变（草稿仍为草稿，已发布仍为已发布）；「保存并发布」将草稿状态覆盖为已发布。

3.3.3 表单校验规则（依次执行）

校验不通过时通过提示对应错误文案，不关闭弹窗：

  1. 练习名称必填 →「请输入练习名称」
  2. 试卷必选 →「请选择试卷」
  3. 练习开放时间必须完整填写 →「请填写完整的练习开放时间」
  4. 结束时间 > 开始时间 →「结束时间必须大于开始时间」
  5. 新增时开始时间 > 当前时间 →「开始时间必须大于当前时间」（编辑时若已开始的练习不可修改开始时间）
  6. 题目总数 > 0 →「请输入有效的题目总数」
  7. 总数 ≤ 试卷题目总数 →「题目总数不能超过试卷题目总数（N 道）」
  8. 至少选一种题型 →「请至少选择一种题目类型」
  9. 各题型之和 = 总数 →「各题型数量之和需等于题目总数」
  10. 各题型数量 ≤ 试卷对应类型上限 →「不能超过试卷单选/多选/判断题数量（N 道）」
  11. 练习人员必填，须至少添加一个人员或部门 →「请至少添加一个人员或部门」

4. 交互设计

4.1 页面加载交互

  • 首次进入「模拟练习」Tab 时自动加载列表数据
  • 与「专业题目练习」Tab 之间切换不重新请求数据（前端 Tab 切换）

4.2 列表交互

  • 行操作即时生效：发布/编辑/删除操作完成后立即刷新列表

4.3 弹窗交互

  • 弹窗采用右侧表单页形式，打开时左侧列表隐藏，左上角返回箭头切换回列表
  • 关闭弹窗仅通过：点击左上角返回箭头 / 点击底部「返回」按钮、保存成功后自动返回列表
  • 编辑模式打开时：
    • 草稿状态：全部字段可编辑，底部显示「返回」「保存」「保存并发布」三个按钮
    • 未开始状态：全部字段可编辑，试卷可选可改，底部显示「返回」「保存设置」两个按钮
    • 进行中状态：试卷选择器、练习名称、题目数量、开始时间区域置灰禁用；结束时间仅可延长（下限取当前时间和开始时间的较大值）；底部显示「返回」「保存设置」两个按钮
    • 非草稿状态点击「保存设置」后弹出确认弹窗二次确认「确定要保存当前设置吗？」，确认后才执行保存
  • 题型复选框勾选/取消勾选实时更新底部汇总显示

4.4 操作反馈

  • 操作成功：提示，3 秒后自动消失
  • 操作失败/校验不通过：提示
  • 删除 / 保存设置（非草稿编辑）：弹出浏览器原生确认弹窗

5. 异常处理

异常场景处理方式
数据加载失败列表显示「暂无数据」
表单校验失败提示具体错误信息，保留弹窗和已填内容
网络异常操作静默失败（当前为前端模拟数据）
无数据列表区域显示「暂无数据」
